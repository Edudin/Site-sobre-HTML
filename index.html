O que é o HTML

(voltar)

HTML ou HyperText Markup Language é uma linguagem de marcação (não de programação), com ele é possível construir e estruturar aplicações web (sites), atuando como esqueleto da página. Ele fornece uma maneira de organizar o conteúdo de uma página, como texto, imagens, vídeos e links, por meio de elementos e tags que definem a estrutura e o significado do conteúdo.

O que são as Tags do HTML

(voltar)

As tags do HTML são elementos fundamentais da linguagem de marcação de hipertexto (HTML), que é usada para criar páginas da web. As tags são estruturas delimitadas por colchetes ("<" e ">" ) que envolvem o conteúdo e fornecem instruções sobre como esse conteúdo deve ser exibido ou processado pelo navegador da web.

Quais são as Tags do HTML

(voltar)
  • html : Define o início e o fim de um documento HTML.
  • head : Contém metadados, como o título da página, links para folhas de estilo e scripts, entre outros.
  • title : Define o título da página exibido na barra de título do navegador ou na guia.
  • body : Contém todo o conteúdo visível da página, como texto, imagens, vídeos e outros elementos.
  • h1, h2, h3, h4, h5, h6: São cabeçalhos de diferentes níveis, do mais importante (h1) ao menos importante (h6) .
  • p : Define um parágrafo de texto.
  • a : Cria um link para outra página da web, um arquivo ou uma âncora em uma página.
  • img : Insere uma imagem na página.
  • ul : Define uma lista não ordenada.
  • ol :Define uma lista ordenada.
  • li : Define um item em uma lista (ul ou ol).

Atributos das Tags

(voltar)

Temos alguns atributos que podem ser utilizados em tags específicas. Alguns exemplos deles são:

  • id: Identifica exclusivamente um elemento em um documento HTML. É frequentemente usado para referenciar o elemento em CSS ou JavaScript.
  • class: Define uma ou mais classes que podem ser aplicadas a um elemento. Classes são usadas para aplicar estilos CSS ou para selecionar elementos usando JavaScript.
  • href: Especifica o URL de destino para links e âncoras.
  • src: Define o caminho para o recurso (geralmente uma imagem, áudio ou vídeo) que será incorporado na página.
  • alt: Fornece um texto alternativo que é exibido quando a imagem referenciada não pode ser carregada.
  • title: Fornece um texto de título que pode ser exibido como uma dica de ferramenta quando o usuário passa o mouse sobre o elemento.
  • target: Especifica onde abrir o link, como em uma nova janela do navegador ou na mesma janela.
  • style: Permite definir estilos CSS diretamente no elemento HTML.

Texto em HTML

(voltar)

Em HTML, o texto é a parte principal do conteúdo que os usuários veem em uma página da web. Ele pode incluir palavras, frases, parágrafos e outros elementos textuais. Aqui estão algumas características importantes do texto em HTML:

  • Parágrafos: Os parágrafos são blocos de texto separados que são definidos pela tag p. Eles são usados para organizar o texto em unidades lógicas e visualmente distintas.
  • Cabeçalhos: Os cabeçalhos são usados para dar destaque a seções importantes do conteúdo. Eles variam em tamanho e importância de h1 (o maior e mais importante) a h6 (o menor e menos importante).
  • Links: Os links permitem que os usuários naveguem entre páginas da web e recursos relacionados. Eles são criados com a tag a e têm um atributo href que especifica o destino do link.
  • Listas: As listas são usadas para organizar itens relacionados. ul é uma lista não ordenada (com marcadores), ol é uma lista ordenada (com números ou letras) e li define cada item da lista.
  • Texto Em Negrito e Itálico: strong é usado para enfatizar texto e é exibido em negrito por padrão, enquanto i é usado para enfatizar texto e é exibido em itálico por padrão.
  • Texto Sublinhado, Subscrito e Sobrescrito: u é usado para sublinhar o texto, sup é usado para texto sobrescrito (como expoentes) e sub é usado para texto subscrito (como índices).
  • Comentários: Os comentários são usados para adicionar notas ou descrições no código HTML que não são exibidas na página da web. Eles são delimitados por !-- e --.

Listas Ordenadas e não Ordenadas

(voltar)

Listas Ordenadas: As listas ordenadas são usadas para organizar itens em uma sequência específica. Cada item da lista é precedido por um número ou uma letra, indicando sua posição na sequência. Os elementos chave são:

  1. ol define a lista ordenada.
  2. li define cada item da lista ordenada.

Listas Não Ordenadas: As listas não ordenadas são usadas para organizar itens em uma ordem sem importância ou sequência específica. Cada item é precedido por um marcador, geralmente um ponto, um quadrado ou um círculo. Os elementos chave são:

  • ul define a lista não ordenada.
  • li define cada item da lista não ordenada.

Links em HTML

(voltar)

Links: Os links em HTML são usados para criar referências a outras páginas da web, recursos online, arquivos locais ou âncoras em uma página. Os elementos chave são:

  • a: Define um hyperlink, ou link.
  • href: Especifica o destino do link, que pode ser um URL absoluto, um URL relativo, um ID de âncora, ou um endereço de e-mail.
  • target: Especifica onde abrir o destino do link, como em uma nova janela (_blank) ou na mesma janela (_self).
  • title: Fornecer um título opcional que aparece quando o cursor do mouse é movido sobre o link.
